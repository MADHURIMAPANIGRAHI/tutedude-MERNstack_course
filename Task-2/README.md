
# Laundry Wallah / Laundry Mart Webpage
    This project is a basic HTML-only webpage developed as part of the TuteDude MERN Stack course. The goal of the project was to create a functional, structured layout for a laundry service business using core HTML elements.

# Project Description
    The webpage serves as an online presence for "Laundry Wallah / Laundry Mart," showcasing their services, pricing, and a booking system for customers.

# Features Included:
    Header: Clear branding for the business.
    About Section: A detailed introduction to the services provided.
    Image Integration: Visual representation of the laundry service.
    Services List: An unordered list displaying various garment care options.
    Price Table: A bordered table showing service costs (Washing, Drying, Ironing, etc.).
    Booking Form: A functional form with required fields for Name, Email, and Phone number.
    Footer: Official contact information for the business.

# Technologies Used
    HTML5: Used for the entire structure and content of the site.
