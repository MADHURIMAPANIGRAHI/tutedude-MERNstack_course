## Features Implemented
- **Body Styling:** Applied a background color to the entire page.
- **Class Selectors:** Styled multiple headings using the `.head` class.
- **ID Selectors:** Used `#bglime` and `#para` for high-specificity styling.
- **Complex Selectors:** Styled specific anchor (`<a>`) tags and list items (`<li>`) using `nth-child` and descendant combinators to avoid using extra IDs or classes.
- **Text Formatting:** Demonstrated the use of `span` for mid-sentence styling.

## Files
- `index.html`: Contains the structure and content.
- `style.css`: Contains all the styling logic.

## How to View
Open `index.html` in any modern web browser.
