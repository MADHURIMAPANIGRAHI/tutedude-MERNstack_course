# Laundry Landing Page

This is a simple laundry website page made using HTML and CSS.

I made:
- A navbar with logo, menu items, and username
- A hero section with text, button, and image

## What I Learned

- How `display:inline` works in navbar elements
- Difference between `inline` and `inline-block`
- How to use `100vh` to fit the screen height
- How to stop the page from scrolling

I did not use flexbox because it was not allowed in the assignment.

## How to Run

1. Open the folder
2. Open `index.html` in browser
